A FUNDADORA

"Escrevendo que eu existo. A minha forma de existir no mundo é através da escrita."

Fernanda Pinho é jornalista com sólida trajetória no impresso (Revista Imobiliari, Líder, Ponteio). Ela entendeu que a arquitetura exige uma linguagem única: técnica para o profissional, poética para o cliente.

Na Entrelaços, unimos o rigor da apuração jornalística à velocidade do digital. Não fazemos apenas posts; documentamos legados.

Nossos Serviços

Gerenciamento Total

Serviço por meio do qual cuidamos integralmente das redes sociais de um cliente desenvolvendo planejamento estratégico de conteúdo, copy, escolha de imagens, arte gráfica, organização de feed, roteiros para vídeos, programação de postagens e outras demandas específicas.

  • Planejamento Estratégico
  • Copy Jornalístico
  • Curadoria de Imagens
  • Edição de Reels
  • Tráfego Pago

Consultoria

A consultoria é uma contratação pontual para um encontro (presencial ou on-line) com duas horas de duração na qual trabalharemos para resolver uma demanda específica. Esta demanda é diagnosticada por meio de um briefing previamente enviado ao cliente.

  • Diagnóstico via Briefing
  • Análise de Posicionamento
  • Definição de Linha Editorial
  • Treinamento de Equipe
  • Online ou Presencial

Script para Vídeos

Visando auxiliar nosso mercado na entrega de vídeos que informam, impactam e emocionam, produzimos roteiros para gravação bem como textos em diferentes formatos para serem utilizados como narração desses vídeos.

  • Scripts para YouTube
  • Roteiros para Reels/TikTok
  • Storytelling de Obra
  • Textos para Narração
  • Direção de Cena
